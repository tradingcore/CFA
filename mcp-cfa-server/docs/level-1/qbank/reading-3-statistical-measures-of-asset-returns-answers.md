Which security has the highest level of relative risk as measured by the coefficient of variation?

A) X.

## Explanation

The coefficient of variation, $CV =$ standard deviation / arithmetic mean, is a common measure of relative dispersion (risk). $CV_{X} = 0.7 / 0.9 = 0.78$; $CV_{Y} = 4.7 / 1.2 = 3.92$; and $CV_{Z} = 5.2 / 1.5 = 3.47$.
